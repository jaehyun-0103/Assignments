# 1. 프레임워크프로그래밍

# DI + JDBC

기능
1. 전체 학생 정보 출력
2. 개인 학생 정보 출력
3. 개인 정보 학생 생성
4. 개인 정보 학생 삭제
5. 개인 정보 학생 수정
6. 테스트 케이스 3개 작성

---

# MVC + MyBatis

기능
1. 전체 학생 정보 출력 (/member/list)
2. 개인 학생 정보 출력 (/member/read?id=hansol)
3. 학생 정보 수정 (/member/modify?id=hansol)
4. Junit 단위 테스트 3가지
5. 개인 정보 삭제
6. Rest api (/member/rest/{id} 제공 - GET Method (JSON 형식)
7. Rest api (/member/rest/ 제공 - POST Method (JSON 형식)
